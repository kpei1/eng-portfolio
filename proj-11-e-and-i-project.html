Electronics & Instrumentation Design Project

Testing of both the project simulation and physical circuit

Objective: Design and create a circuit that will alert the user if a given temperature threshold has been exceeded. An LED must turn on once the temperature threshold has been crossed, and in order to save energy, the LED must have a "check status" button so that it is not on constantly. The LED should stay on, even if the temperature drops back below the threshold, until the user presses the reset button.

Design: We used a potentiometer to allow the user to adjust the threshold temperature of the device. Our TMP36 temperature sensor was connected to an op-amp. The temperature sensor and the op-amp were configured so that the op-amp would output 5V while the temperature was below the threshold, and 0V while the temperature was above the threshold. The output from our op-amp went to a 555 Timer chip, which remembered whether the threshold had been exceeded. The LED was connected to the output of the 555 Timer, and the reset button was connected to the 555 Timer's reset leg.
