"UR5 Robot Control"

Objective: Control the UR5 robotic arm using both forward and inverse kinematics to perform a pick-and-place routine.

Design: We used MATLAB to write functions for our forward and inverse kinematics, as well as all the needed helper functions, such as Jacobian calculations, etc. We were not allowed to use MATLAB's predefined functions in the robotics toolbox, so we wrote all of our own functions. We used ROS with Gazebo to test the UR5 virtually before moving to physical tests. For extra credit, we programmed the robotic arm to draw our professor's face using inverse kinematics.
